# 素材说明

本目录存放课程作业可使用的原创素材。素材为像素风平台跳跃游戏资源，适合搭配 Pygame 使用。

## 图片素材

| 文件 | 用途 | 建议使用方式 |
|---|---|---|
| `images/background_day.png` | 白天关卡背景 | 第 1 关背景，铺满窗口 |
| `images/background_cave.png` | 地下洞穴背景 | 第 2 关背景，铺满窗口 |
| `images/background_sky_castle.png` | 天空城堡背景 | 第 3 关背景，铺满窗口 |
| `images/cover_concept.png` | 项目封面示意图 | 可放在报告或 PPT 中 |
| `images/player_right.png` | 玩家向右图片 | 玩家朝右移动时显示 |
| `images/player_left.png` | 玩家向左图片 | 玩家朝左移动时显示 |
| `images/tile_ground.png` | 地面砖块 | 按地图格子重复绘制 |
| `images/coin.png` | 金币 | 玩家收集物 |
| `images/enemy_mush.png` | 蘑菇敌人 | 左右巡逻敌人 |
| `images/enemy_bird.png` | 飞行敌人 | 第 2、3 关敌人 |
| `images/goal_flag.png` | 终点旗 | 通关目标 |
| `images/question_block.png` | 问号砖块 | 顶击前显示 |
| `images/used_block.png` | 已触发砖块 | 问号砖块触发后显示 |
| `images/life_mushroom.png` | 加命奖励 | 问号砖块随机奖励 |

## 音频素材

| 文件 | 用途 | 触发时机 |
|---|---|---|
| `audio/theme_loop.wav` | 背景音乐 | 游戏开始后循环播放 |
| `audio/jump.wav` | 跳跃音效 | 玩家起跳或踩敌人反弹时播放 |
| `audio/coin.wav` | 金币音效 | 收集金币或获得金币奖励时播放 |
| `audio/hurt.wav` | 受伤音效 | 玩家碰到敌人受伤时播放 |

## 素材风格说明

- 风格：原创像素风、明亮、适合横版平台跳跃游戏。
- 尺寸：角色和敌人适合缩放到 32x32 或 32x40；地砖、金币、奖励适合 32x32 网格；背景适合 960x640 窗口。
- 授权：项目内生成素材，仅用于本课程作业和学习展示。
- 注意：不要把素材描述为官方《超级马里奥》素材，答辩时建议说“参考经典平台跳跃玩法的原创素材”。
